Operating Systems Terminology
TERM DEFINITION
operating system The software that controls the overall operation of a computer including file management, the user interface, and the program execution environment.
job queue A storage organization in which jobs are ordered in first-in, first-out fashion.
FIFO first-in, first-out, objects that are removed from the queue in the order from which they arrived.
interactive processing Processing method which allowed a program being executed to carry on a dialogue with the user through remote terminals.
time-sharing Operating systems that provide service to multiple users at the same time.
multiprogramming Technique in which time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
application software Programs for performing tasks particular to the machine's utilization, such as spreadsheets, database systems, desktop publishing systems, accounting systems, program development software, and games.
system software Software that provides the infrastructure that the application software requires.
utility software Programs for performing activities that are fundamental to computer installations but not included in the operating system.
graphical user interface (GUI) The objects to be manipulated, such as files and programs, are represented pictorially on the display as icons.
shell Older user interfaces that communicated with users through textual messages using a keyboard and monitor screen.
kernel The internal part of an operating system that contains software components that perform the very basic functions required by the computers installation.
device driver The software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
file manager The software component whose job is to coordinate the use of the machine's mass storage facilities.
memory manager The software component whose job is to coordinate the use of the machine's main memory.
virtual memory A large fictional memory space created by paging when the memory manager will rotate programs and data back and forth between main memory and mass storage.
boot strapping A procedure that is performed by a computer each time it is turned on when the operating system is transferred from mass storage into main memory.
read-only memory (ROM) Special nonvolatile memory cells that can be read but not altered where the CPU expects to find its initial program.
deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
spooling Technique of holding data for output at a later but more convenient time.
super user (or administrator) The person with highly privileged access to the operating system, which allows them to alter settings within the operating system, modify critical software packages, adjust privileges granted to other users, and perform a variety of other maintenance activities that are denied normal users.
multi-factor authentication A process of securing computer accounts with more than one piece of evidence to gain access.
