Network & The Internet
TERM DEFINITION
Network Classifications
  • LAN: Local area network- a collection of computers in a building or buidling complex.
  • MAN: Metropolitan area network- intermediate size, such as one spanning a local community.
  • WAN: Wide area network- links machines over a greater distance, such as neighboring cities or the opposite side of the world.
Network models
  • Bus network: Machines are connected to a common communication line called a bus.
  • Star network: One machine serves as the central focal point to which all the others are connected.
Protocols
  • CMSA/CD Ethernet: Carrier Sense, Multiple Access with Collision Detection- This protocol dictates that each message be broadcast to all the machines on the bus.
  • CSMA/CA Wifi: Carrier Sense, Multiple Access with Collision Avoidance- Policies adobted by wireless networks to avoid collisions rather than trying to detect them.
Combining networks
  • Repeater: Device that passes signals back and forth between the two original buses without considering the meaning of the signals.
  • Bridge: It connects two buses, and looks at the destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side.
  • Switch: A bridge with multiple connections, allowing it to connect several buses rather than just two.
  • Router: Special purpose computers used for forwarding messages.
  • Gateway: A point at which one network is linked to an internet.
Client/server model
  • P2P: Peer-to-peer- Involves processes that provide service to and receive service from each other.
  • Distributed system- They consist of software units that execute as processes on different computers.
  • Cloud computing- Huge pools of shared computers on the network can be allocated for use by clients as needed.
Internet Addressing
  • Tier 1: Very high-speed, high capacity, international WAN's. Thought of as the backbone of the internet.
  • Tier 2: ISP's that tend to be more regional in scope and less potent in their capabilities.
  • Tier 3: An independent internet, sometimes called an intranet.
IP address
  • ICANN: Internet Corporation for Assigned Names and Numbers- Nonprofit corportation established to coordinate the internet's operation.icann.org
  • DNS: Domain name system- Internet wide directory system.
  • TLD: Top-level domain- Suffixes following domain names that reflect the domain's classification.
  • Country-code TLD's: Suffixes created for specific countries. worldstandards.eu
Internet protocols
  • FTP: File Transfer Protocol- Application for listing and copying files across the network.
  • HTTP: Hyper Text Transfer Protocol- The protocol of transferring hypertext documents between browsers and webservers.
  • SMTP: Simple Mail Transfer Protocol- Defines a way that two computers on the network may interact when transmitting an email message from one host to the other.
World Wide Web
  • URL: Uniform Resource Locator- A unique address given to each document with information needed by a browser to contact the proper server and request the desired document.
  • Client-side activities: Activities if they are performed by the client.
  • Server-side activities: Activities if they are performed by the server.
Internet four-layer software
  • Application layer: Consists of those software units such as clients and servers that use internet communication to carry out their tasks.
  • Transport layer: Accept messages from the application layer and ensure that the messages are properly formatted for transmission over the internet.
  • Network layer: Decide in which direction a packet should be sent at each step along the packet's path through the internet.
  • Link layer: Has the responsibility of transferring the packet.
Internet is:
  • Packet-switching network: A system through which digital data is sent by breaking the data into blocks of bits called packets.
  • Fault tolerant: Cumulative result of the features from transporting, routing, and forwarding packets is an internet that is tolerant of many types of faults.
  • Redundant: More than one way to route data.
Cybersecurity
  • Malware: Malicious software.
  • Virus: Software that infects a computer by inserting itself into programs that already reside in the machine.
  • Worm: An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itself to other computers.
  • Trojan horse: Program that enters a computer system disguised as a desireable program.
  • Spyware: Software that collects information about activities at the computer on which it resides and reports that information back to the instigator of the attack.
  • Phishing: Technique of obtaining information explicitly by simply asking for it.
  • DDoS attack: Distributed denial of service- When the work of generating the attacks is distributed over many hosts.
Protection
  • Firewall: Program used to filter traffic passing through a point in the network.
  • Spam filter: Firewalls designed to block unwanted email.
  • Proxy server: Software unit that acts as an intermediary between a client and a server with the goal of shielding the client from adverse actions of the server.
  • Antivirus software: Software used to detect and remove the presence of known viruses and other infections.
  • Cryptography: The study of sending and receiving messages securely in the presence of adversaries.
  • Public key encryption: Techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure.
